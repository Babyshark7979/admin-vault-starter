# 🛠️ Hướng dẫn Setup

## Bước 1 — Mở vault trong Obsidian

1. Mở Obsidian
2. Chọn **"Open folder as vault"**
3. Trỏ vào thư mục `admin-vault-starter` này
4. Bấm **Open**

## Bước 2 — Cài plugin cần thiết (trong Obsidian)

Vào Settings → Community Plugins → Turn off Safe Mode → Browse

Cài 3 plugin này:
- **Templater** — dùng template có biến ngày tháng tự động
- **Dataview** *(tùy chọn)* — xem danh sách file theo trạng thái
- **Minimal Theme Settings** *(tùy chọn)* — giao diện gọn hơn

## Bước 3 — Thiết lập Templater

1. Settings → Templater
2. Template folder: `Templates`
3. Bật **Trigger Templater on new file creation**

## Bước 4 — Test thử

1. Tạo file mới trong `Inbox/`
2. Đặt tên: `2026-07-05 Công văn thử nghiệm`
3. Dùng template `Templates/nhan-viec.md` để điền thông tin
4. Sửa xong → move sang `Xong/`
