# ⚡ Hướng dẫn nhanh — Dùng ngay trong 10 phút

> Đọc file này trước. Mọi thứ khác đọc sau khi cần.

---

## Setup lần đầu (làm 1 lần duy nhất)

1. Tải Obsidian về máy tại **obsidian.md**
2. Mở Obsidian → chọn **"Open folder as vault"**
3. Trỏ vào thư mục này → bấm **Open**
4. Xong. Vault đã sẵn sàng.

---

> Xem sơ đồ đầy đủ: [[WORKFLOW]] | Prompt AI: [[HUONG-DAN-ANTIGRAVITY]]

## Làm việc hàng ngày — 5 bước

### 1️⃣ Nhận file → vào Inbox

Khi nhận tài liệu cần xử lý:
- Copy file vào thư mục `Inbox/`
- Đặt tên: `2026-07-05 Tên tài liệu` (có ngày ở đầu)

### 2️⃣ Mở template nhận việc

- Trong Obsidian, bấm vào [[Templates/nhan-viec|Template nhận việc]]
- Tạo file mới, copy nội dung template vào
- Điền: ai giao, deadline, cần làm gì

### 3️⃣ Xử lý tài liệu

- Move file sang `Đang làm/`
- Mở [[Templates/checklist-sua-van-ban|Checklist sửa văn bản]]
- Làm theo từng ô checklist từ trên xuống

### 4️⃣ Nhờ AI kiểm tra lại

- Mở **Antigravity**
- Copy đoạn văn cần kiểm tra
- Xem [[HUONG-DAN-ANTIGRAVITY|hướng dẫn prompt]] hoặc paste prompt sau:

> *"Hãy sửa lỗi chính tả và lỗi dấu tiếng Việt trong đoạn văn sau. Chỉ sửa lỗi sai rõ ràng, không thay đổi nội dung. Liệt kê những gì đã sửa."*

- Paste văn bản của bạn vào sau prompt → gửi
- Đọc kết quả → tự quyết áp dụng hay không

### 5️⃣ Xong → move sang Xong/

- Lưu file bản cuối, thêm chữ "FINAL" vào tên
- Move file sang thư mục `Xong/`
- Nộp bài theo yêu cầu

---

## Khi bị kẹt — hỏi AI thế này

Mở Antigravity và nhắn:

> *"Tôi đang làm [mô tả ngắn việc đang làm]. Tôi đang bị kẹt ở [vấn đề cụ thể]. Giúp tôi với."*

AI sẽ hỏi thêm hoặc gợi ý hướng giải quyết.

---

## Thư mục — dùng để làm gì

| Thư mục | Dùng khi nào |
|---|---|
| `Inbox/` | File mới nhận, chưa xử lý |
| `Đang làm/` | Đang xử lý / sửa |
| `Xong/` | Đã xong, đã nộp |
| `Templates/` | Lấy mẫu ra dùng (đừng sửa file gốc) |
| `Ghi chú/` | Ghi ý tưởng, ghi nhớ tự do |
| `logs/` | Nhật ký ngày — ghi ngắn hôm nay làm gì |

---

## 3 quy tắc quan trọng nhất

1. **File mới → Inbox trước**, không vứt lung tung
2. **Xong việc → move sang Xong/**, đừng để đọng trong Đang làm
3. **Không xóa file gốc** — luôn giữ bản gốc khi sửa
