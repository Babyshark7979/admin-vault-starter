# 🗂️ Admin Vault Starter — Bộ não làm việc hành chính

> Phiên bản đơn giản hóa từ hệ thống Second Brain thực tế — dùng ngay, chỉnh sửa thoải mái.

---

## Cấu trúc thư mục

```
admin-vault-starter/
│
├── 📥 Inbox/          ← File mới đến, chưa xử lý
├── 🔧 Đang làm/       ← Đang review hoặc sửa
├── ✅ Xong/           ← Đã hoàn thành, lưu để tra cứu
├── 📋 Templates/      ← Mẫu dùng lại
├── 📝 Ghi chú/        ← Ghi chú, ý tưởng, quyết định
└── logs/              ← Nhật ký công việc theo ngày
```

---

## Nguyên tắc cốt lõi (giữ nguyên, đừng phá)

1. **Inbox trước, xử lý sau** — file nào mới đến đều vào Inbox trước, không vứt lung tung
2. **Một việc, một file** — đừng gộp nhiều tài liệu vào 1 file; dễ tìm sau này
3. **Xong thì move** — khi hoàn thành việc gì → move file sang thư mục Xong
4. **Dùng template** — không tạo mẫu mới khi đã có sẵn trong Templates/
